gwt:import-sources

Full name:

net.ltgt.gwt.maven:gwt-maven-plugin:1.0-rc-6:import-sources

Description:

Copies all source files as if they were resources (with no filtering or relocation).

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: process-classes.

Required Parameters

Name Type Since Description
outputDirectory File - (no description)
Default value is: ${project.build.outputDirectory}.

Optional Parameters

Name Type Since Description
encoding String - The character encoding scheme to be applied when copying files.
Default value is: ${project.build.sourceEncoding}.
User property is: encoding.
overwrite boolean - Overwrite existing files even if the destination files are newer.
Default value is: false.
User property is: maven.resources.overwrite.
