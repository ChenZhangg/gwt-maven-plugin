gwt:import-test-sources

Full name:

net.ltgt.gwt.maven:gwt-maven-plugin:1.0-SNAPSHOT:import-test-sources

Description:

Adds all test-source directories as resources, and unpacks and add as resource each test dependency with <type>java-source</type>.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: generate-resources.

Required Parameters

Name Type Since Description
importedTestSourcesTargetDirectory File - Location where java-source dependencies will be unpacked.
Default value is: ${project.build.directory}/generated-resources/gwt-test-sources.

Parameter Details

importedTestSourcesTargetDirectory:
